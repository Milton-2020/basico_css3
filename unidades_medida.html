Unidades de medida

Las medidas CSS se emplean, entre otras, para definir la altura, anchura y márgenes de los elementos y para establecer el tamaño de la letra del texto. Todas las medidas se indica como un valor numérico entero o decimal seguido de una de medida (si espacios en blancos ente el número y la unidad)

CSS divide las unidades de medida en dos grupos absolutas y relativas.

Si el valor es 0, la unidad de medida es opcional. Si el valor es distinto de cero y no se indica ninguana unidad, la medida se ignora completamente. Algunas propiedades permiten definir valores negativos, aun que habitualmente sus valores son positivos.

Unidades relativas.

Las unidades relativas son más flexibles que las unidades absolutas por que se adaptan más fácilmente a los diferentes medios. A continuacion se muestra la lista de unidades de medida relativas y la referencia que se toma para determinar su valor real.

  1. em, (no confundir con la etiqueta <em> de HTML) Aun que no es una definción exacta, el valor de un 1em se puede aproximar por la anchura de la letra "M".
  2. ex, relativa respecto de la altura de la letra "x" (minuscula)
  3. px (píxel), relativa respeto de la pantalla del usuario.

Si se utiliza la tipografia de 12 puntos, 1em equivale a 12 puntos. El valor 1ex se puede aproximar por 0.5em.

En el siguiente ejemplo, se indica que el tamaño de la letra del texto de la página debe de ser el 90% del tamaño por defecto.

                    
                        body{ font-size: 0.9;}

Cuando el valor decimal de una medida es inferiro a 1, se puede omitir el 0 de la izquierda, por lo que el código anterior es equivalente al código siguiente:

                    
                        body{ font-size: .9;}

Unidades absolutas.

Las unidades absolutas definen la medida de forma completa, ya que sus valores reales no se calculan a partir de otros valores de referencia, sino que son directamente los valores indicados. A continuación se muestra la lista completa de unidades absolutas definidas por CSS y su significado.

  • in, del ingles "inches", pulgadas(1 pulgada son 2.54 centímetros)
  • cm, centímetros.
  • mm, milímetros.
  • pt, puntos(un punto equivale a una pulgada/73, es decir, unos 0.35 milímetros)
  • pc, picas (1picas equivale a 12 puntos, es decir, unos 4.23 milímetros)

A continuación se muestran ejemplos de utilización de unidades absolutas.

                    
                        body{ margin: 0.5in;}
                        h1{ line-height: 2cm;}
                        p{ word-spacing: 4mm}
                        a{ font-size: 12pt}
                        span{ font-size: 1pc}

De todas las unidades absolutas, la única que se utiliza con cierta frecuencia es la de los puntos(pt). El motivo es que se trata de una unidad preferida para indicar el tamaño de letra de texto para los documentos que se vana a imprimir, es decir, para, el medio print de CSS.

Porsentajes.

CSS define otra unidad relativa de medida basada en los porcentajes y siempre esta referenciado a otra medida.

Los porcentajes se pueden utilizar por ejemplo para establecer el valor del tamaño de letra de los elementos.

                    
                        body{font-size: 1em;}
                        h1{font-size: 200%;}
                        H2{font-size: 150%;}

Los tamaños establecidos para <h1> y <h2> mediante las reglas anteriores, son equivalentes a 2em y 1.5em respectivamente, por lo que es más habitual definirlos mediante em.

Los porcentajes tambien se utilizan para defienir la anchura de los elementos.

Regla CSS.

                    
                        div#contenido{width: 600px;}
                        div.principal{width: 80%;}

Código HTML.

                    
                        <div id="contenido">
                            <div class="principal">
                                .....
                            </div>
                        </div>

En el ejemplo anterior, la referencia del valor 80% es la anchura de elemento padre. Por lo tanto, el elemento <div class="principal"> tiene un ancho de 80% x 600px = 480px

Recomendaciones.

En general, se recomienda el uso de unidades relativas siempre que sea posible, ya que mejora la accesibilidad de la página y permite que los documentos se adapten fácilmente a cualquier medio y dispositivo.

Normalmente se utiliza píxel y porcentaje para definir el layout del documento(básicamente, la anchura de las columnas y elementos de las páginas.) y em y porcentaje para el tamaño de la letra de los textos.
