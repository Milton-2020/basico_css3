Selectores

Selector universal

Se utiliza para seleccionar todos los elementos de una página.

Ejemplo:

 
                    
                        * {
                            margin: 0;
                            padding: 0;
                        }

Selector de tipo o etiqueta

Selecciona todo los elementos de la página cuyo etiqueta HTML coincide con el valor del selector.

Ejemplo selecionando mediante la etiqueta p:

                    
                        p{
                            *****
                        }

Ejemplo aplicando estilo a los titulos (h...) y a los parrafos (p)

                    
                        h1{
                            color: red;
                        }
                        h2{
                            color: blue;
                        }
                        p{
                            color: black;
                        }

CSS permite agrupar todas las reglas individuales en una sola regla con el selector múltiple. Para ello, se incluye todos los selectores separados por una coma (,).

Ejemplo de agrupacion:

                    
                        h1, h2, h3{
                            color: #8A8E27;
                            font-weight: normal;
                            font-family: Areal, Helvetica, sans-serif;
                        }

Selecetores descendente.

Los selectores decendentes siempre estan formados por dos partes o mas partes separadas entres sí por espacios en blanco.La última parte indica el elemento sobre el que se aplican los estilos y todas las partes anteriores indica el lugar en el que se tiene que encontrar ese elemento par que los estilos se apliquen realmente.

El selector del siguiente ejemplo selecciona todos los elementos span de la página que se encuentra dentro de un elemento p.

                    
                        p span{
                            color: red;
                        }
                        h3 span{
                            color: red;
                        }

Nota: No confundir el selector decendente con la combinación de selectores

Combinación de selectores.

El estilo se aplica a todos los elementos (p, a, span, em).

                    
                        p, a, span, em{
                            text-decoration: underline;
                        }

Selector decendente.

El estilo se aplica solo alos elemtos em que se encuentren dentro de p a span

                    
                        p a span em{
                            text-decoration: underline;
                        }

Si se emplea el selector decendete con el selctor universal se puede restingir el alcanse de un selector decendente.

El siguiente ejemplo muestra los dos enlaces de color rojo.

                    
                        p a{
                            color: red;
                        }

Codigo HTML

                    
                        <p><a href=""#>Enlace1</a></p>
                        <p><span><a href=""#>Enlace2</a><span></p>

Sin embargo, en el siguiente ejemplo solamente el segundo enlace se muestra de color rojo.

                    
                        p * a{
                            color: red;
                        }

Codigo HTML

                    
                        <p><a href=""#>Enlace1</a></p>
                        <p><span><a href=""#>Enlace2</a><span></p>

La razon es que el selector p * a se traduce como todos los elementos de tipo <a> que se encuentre dentro de cualquier elemento que, a su vez, se encuentre dentro de un elemento de tipo <p>. Como el primer elemento <a> se encuentra directamente bajo un elemento <p>, no se cumple la condición del selector p * a.
