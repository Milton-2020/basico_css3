Selectores

Selector universal

Se utiliza para seleccionar todos los elementos de una página.

Ejemplo:

 
                    
                        * {
                            margin: 0;
                            padding: 0;
                        }

Selector de tipo o etiqueta

Selecciona todo los elementos de la página cuyo etiqueta HTML coincide con el valor del selector.

Ejemplo selecionando mediante la etiqueta p:

                    
                        p{
                            *****
                        }

Ejemplo aplicando estilo a los titulos (h...) y a los parrafos (p)

                    
                        h1{
                            color: red;
                        }
                        h2{
                            color: blue;
                        }
                        p{
                            color: black;
                        }

CSS permite agrupar todas las reglas individuales en una sola regla con el selector múltiple. Para ello, se incluye todos los selectores separados por una coma (,).

Ejemplo de agrupacion:

                    
                        h1, h2, h3{
                            color: #8A8E27;
                            font-weight: normal;
                            font-family: Areal, Helvetica, sans-serif;
                        }

Selecetores descendente.

Los selectores decendentes siempre estan formados por dos partes o mas partes separadas entres sí por espacios en blanco.La última parte indica el elemento sobre el que se aplican los estilos y todas las partes anteriores indica el lugar en el que se tiene que encontrar ese elemento par que los estilos se apliquen realmente.

El selector del siguiente ejemplo selecciona todos los elementos span de la página que se encuentra dentro de un elemento p.

                    
                        p span{
                            color: red;
                        }
                        h3 span{
                            color: red;
                        }

Combinación de selectores.

El estilo se aplica a todos los elementos (p, a, span, em).

                    
                        p, a, span, em{
                            text-decoration: underline;
                        }

Selector decendente.

El estilo se aplica solo alos elemtos em que se encuentren dentro de p a span

                    
                        p a span em{
                            text-decoration: underline;
                        }

Si se emplea el selector decendete con el selector universal se puede restingir el alcanse de un selector decendente.

El siguiente ejemplo muestra los dos enlaces de color rojo.

                    
                        p a{
                            color: red;
                        }

Codigo HTML

                    
                        <p><a href=""#>Enlace1</a></p>
                        <p><span><a href=""#>Enlace2</a><span></p>

Sin embargo, en el siguiente ejemplo solamente el segundo enlace se muestra de color rojo.

                    
                        p * a{
                            color: red;
                        }

Codigo HTML

                    
                        <p><a href=""#>Enlace1</a></p>
                        <p><span><a href=""#>Enlace2</a><span></p>

La razon es que el selector p * a se traduce como todos los elementos de tipo <a> que se encuentre dentro de cualquier elemento que, a su vez, se encuentre dentro de un elemento de tipo <p>. Como el primer elemento <a> se encuentra directamente bajo un elemento <p>, no se cumple la condición del selector p * a.

Selector de clase.

Una de las soluciones más senciallas para aplicar estilos a un solo elemento de la página consiste en utiliza el atributo class de HTML dobre ese elemento para indicar directamente la regla CSS que se le deve aplicar.

Principal característica de este selector es que en una misma página HTML varios elementos diferentes pueden utilizar el mismo valor en el atributo class.

Codigo HTML

                    
                        <body>
                            <p class="ejemplo">Ejemplo con la etiqueta p y el atributo class.</p>
                            <p>Ejemplo sin el atributo class.</p>
                            <p>Otro ejemplo sin el atributo class.</p>
                        </body>

Para que el el navegador no confunda este selector con otros tipos de selectores, se prefija el valor del atributo class con un punto (.)

Selector de clase.

                    
                        .ejemplo{
                            color: red; 
                        }

El selector .ejemplo se interpreta com "Cualquier elemento de la página cuyo atributo class sea igual a ejemplo", por lo que solamente el primer párrafo cumple esa condición segun el ejemplo antes expuesto.

                    
                        <body>
                            <p class="ejemplo">Ejemplo con la etiqueta p y el atributo class.</p>
                            <p>Ejemplo sin el atributo class en la etiqueta p.<a class="ejemplo">pero si en la etiqueta a</a></p>
                            <p>Otro ejemplo sin el atributo class en la etiqueta p. <em class="ejemplo">pero si en la etiqueta em</em></p>
                        </body>

¿Cómo es posible aplicar estilos solamente al párrafo cuyo atributo class sea igual a ejemplo? Combinando el selector de tipo y el selectro de classe, se obtiene un selector mucho más específico

Ejemplo:

                    
                        p.ejemplo{
                            color: red;
                        }

EL selector p.ejemplo se interpreta como "aquellos elementos de tipo <p> que dispongan de un atributo class con el valor ejemplo".

Nota: no debe confundirse el selector de clase con los selectores anteriores.

Todos los elementos de tipo "p" con el atributo class="aviso".

                    
                        p.aviso{
                            color: red;
                        }

Todos los elementos con el atributo class="aviso" que estén dentro de cualquier elemento de tipo "p".

                    
                        p .aviso{
                            color: red;
                        }

Código HTML.

                    
                        <p>este es un ejemplo<a class="aviso"></a>el el atributo class</p>
                        <p>este es un ejemplo<em class="aviso">con el atributo class</em></p>

Todos los elementos "p" de la página y todos los elementos con atributo calss="aviso" de la página.

                    
                        p, .aviso{
                            color: red;
                        }

Por último, es posible aplicar los estilos de varias clases CSS sobre un mismo elemento. La sintaxis es similar, pero los diferentes valores del atributo class se separan con espacios en blanco.

Código HTML

                    
                        <p class="especial ejemplo aviso">Párrafo de texto de ejemplo</p>

Al párrafo anterior se le aplica los estilos definidos en las regals .especial, ejemplo y aviso.

                    
                        .especial{
                            color: red;
                        }
                        .ejemplo{
                            font-weight: blod;
                        }
                        .aviso{
                            font-size: 15px;
                        }

Al párrafo anterior se le aplica los estilos definidos en las reglas .especial, .ejemplo y .aviso, por lo que el siguiente ejemplo, el texto del párrafo se vería de color rojo, en negrita y con un tamaño de letra de 15 píxel

Este parrafo es de ejemplo aplicando lo ya antes mencionado.

Si un elemento dispone de un atributo class con más de un valor, es posible utilizar un selector más avanzado.

                    
                        .especial{
                            color: red;
                        }
                        .especial.ejemplo{
                            color: blue;
                        }
                        .ejemplo{
                            font-size: 15px;
                        }
                        .aviso{
                            font-weight: bold;
                        }

en el ejemplo anterior el texto es de color azul y no rojo. El motivo es que se ha utilizar un selector de clase múltiple .especial.ejemplo, que se interpreta como "aquellos elementos de la página que dispongan de un atributo class con al menos los valores especial y ejemplo".

Selector de ID.

El selector de ID permite seleccionar un elementode la página a trevés del valor de su atributo id. Este tipo de selectores sólo selecciona un elemento de la página porque el valor del atributo id no se puede repetir en dos elementos diferentes de una misma página.

La sintaxis de los selectores de ID es muy parecida a la de los selectores de clase, salvo que se utiliza el simbolo numeral (#) en vez del punto (.).

                    
                        #aviso{
                            color: blue;
                        }

Código HTML:

                    
                        <p>Primer párrafo.</p>
                        <p id="aviso">Segundo párrafo.</p>
                        <p>Tercer párrafo.</p>

La pricipal diferencia entre este tipo de selector y el selector de clase tiene que ver con HTML y no con CSS. Como se sabe, en una página. el valor del atributo id debe ser único, de forma que dos elementos diferentes no pueden tener el mismo valor id. Sin embargo, el atributo class no es obligatorio que sea único, de forma que muchos elementos HTML diferenres pueden compartir el mismo valor para su atributo class.

De esta forma, la recomendación general es la de utilizar el selector de ID cuando se quiere aplicar un estilo a un solo elemento específico de la página y utiliza el selector de clase cuando se quiere aplicar un estilo a varios elementos diferentes de la página HTML.

Nota: no debe confundirse el selector de ID con los selectores antes mencionados.

Todos los elemetos de tipo "p" con atributo id="aviso"

                    
                        p#aviso{
                            color: blue;
                        }

Todos los elementos con atributo id="aviso" que estén dentro de cualquier elemento de tipo "p".

                    
                        p #aviso{
                            color: blue;
                        }

Todos los elemetos "p" de la página y todos los elementos con atributo id="aviso" de la página

                    
                        p, #aviso{
                            color: blue;
                        }

Combinación de selectores básicos.

Ejemplos habituales de combinación de selectores.

Solamente selecciona aquellos elementos con una clase="especial" que se encuentre dentro de cualquier elemento con una clase="aviso".

                    
                        .aviso .especial{
                            color: blue;
                        }

Ahora, el selector solamente seleccionara aquellos elementos de tipo <span> con un atributo class="especial" que estén dentro de cualquier elemento de tipo <div> que tenga un atributo class="aviso".

                    
                        div.aviso span.especial{
                            color: blue;
                        }

El selector hace referencia al enlace con un atributo id="inicio" que se encuentre dentro de un elemento de tipo <li> con un atributo class="destacado", que forma parte de una lista <ul> con un atributo id="menuPrincipal"

                    
                        ul#menuPrincipal li.destacado a#inicio{
                            color: blue;
                        }
Ejemplos de selectore.

Selectores avanzados.

Utilizando solamente los selectores básicos de la sección anterior, es posible diseñar prácticamente cualquier página web. No obstante, CSS define otrod selectores más avanzados que permiten simplificar las hojas de estilo.

Desafortunadamente, el navegador internet explorer 6 y sus verciones anteriores no soportan este tipo de selectores avanzados, por lo que su uso no era común hasta hace poco tiempo.

Selectores hijos.

Se trata de un selector similar al selector descendente, pero muy diferente en sus funcionamiento. Se utiliza para seleccionar un elemento que es hijo directo de otro elemento y se indica mediante el signo "mayor que" (>).

                    
                        p > span {
                            color: blue;
                        }

El codigo HTML al que se aplica es el siguiente.

                    
                        <p><span>Texto1</span></p>
                        <p><a href="#"><span>Texto2</span></a></p>

En el ejemplo anterior, el selector p > span se interpreta como "cualquier elemento <span> que sea hijo directo de un elemento <p>", por lo que el primer elemento <span> cumple la condición del selector. Sin embargo, el segundo elemeto <span> no la cumple porque es descendiente pero no es hijo directo de un elemento <p>.

Comparacion de selector decendente y selectro hijo.

                    
                        p a {
                            color: red;
                        }
                        p > a {
                            color: red;
                        }

Codigo HTMl

                    
                        <p><a href="#">Enlaze1</a></p>
                        <p><span><a href="#">Enlace2</a></span></p>

El primer selector es de tipo desendente y por tanto se aplica a todos los elementos <a> que se encuentran dentro de elementos <p>. En este caso, los estilos de este selector se aplican a los dos enlaces.

Por otra parte, el selector de hijos obliga a que el elemento <a> sea hijo directo de un elemento <p>. Por lo tanto, los estilos del selector p > a no se aplican al segundo enlce del ejemplo anterior.

Selector adyacente.

El selecetore adyacente utiliza el signo +.

                    
                        elemento1 + elemento2{
                            color: blue;
                        }

La explicación del comportamiento de este selector no es sencilla, ya qu4e selcciona todos los elementos de tipo elemento2 que cumpla las dos siguientes condiciones:

elemento1 y elemento2 deben ser hermanos, por lo que su elemento padre debe ser el mismo.

elemento2 debe aparecer inmediatamente después de elemento1 en el código HTML de la página.

Ejemplo:

                    
                        h1 + h2 {
                            color: red
                        }

Código HTML

                    
                        <body>
                        <h1>Titlulo1</h1>
                        <h2>Subtítulo</h2>
                        ...
                        <h2>Otro subtítulo</h2>
                        </body>

Los estilos del selector h1 + h2 se aplican al primer elemento <h2> de la página, pero no al segundo elemento.

El elemento padre de <h1> es <body>, el mismo padre que el de los dos elementos <h2>. Así, los dos elementos <h2> cumplen la primera condición del selector adyacente.

EL primer elemento <h2> aparece en el código HTML justo después del elemento <h1>, por lo que este elemento <h2> también cumple la segunda condicion del selector adyacente.

Por el contrario, el segundo elemento <h2> no aparece justo después del elemento <h1>, por lo que no cumple la segunda condición del selector adyacente y por tanto no se le aplica los estilos de h1 + h2.

Selector de atributos.

Selectores de atributos, que permiten seleccionar elementos HTML en función de sus atributos y/o valores de esos atributos.

Los cuatro tipos de slectores de atributos son:

[nombre_atributo], selecciona los elementos que tienen establecido el atributo llamado nombre_atributo, independiente de su valor.

[nombr_atributo=valor], selecciona los elementos que tienen establecido un atributo llamado nombr_atributo con un valor igual a valor.

[nombre_atributo~=valor], selecciona los elementos que tiene establecido un atributo llamado nombr_atributo y al menos uno de los valores del atributo es valor.

[nombre_atributo|=valor], selecciona los elementos que tiene establecido un atributo llamado nombr_atributo y cuyo valor es una serie de palabras separadas con guiones, pero que comienzan con un valor. Este tipo de selector sólo es útil para los atributos de tipo lang que indican el idioma del conttenido del elemento.

Ejemplo de los selectores:

Se muestra de color azul todos los enlaces que tengan un atributo "class", independientemente de su valor.

                    
                        a[class]{
                            color: blue;
                        }

Se muestra de color azul todos los enlaces que apunten al sitio "http://www.ejemplo.com"

                    
                        a[href="http://ejemplo.com"] {
                            color: blue;
                        }

Se muestra de color azul todos los enlaces que tengan un atributo "class" en el que al menos uno de sus valores sea "valor1"

                    
                        a[class~="valor1"]{
                            color: blue;
                        }

selecciona todos los elementos de la página cuyo atributo "lang" sea igual a "en", es decir, todos los elementos en inglés.

                    
                        [lang="en"]{
                            ...
                        }

Selecciona todos los elementos de la página cuyo atributo "lang" empiece por "es", es decir, "es", "es-ES", "es-AR", etc.

                    
                        [lang|="es"]{
                            color: red;
                        }

selcciona todos elementos de la página cuyo atributo "lang"
