Calculo de anchura y altura final

La anchura y la altura de un elemento no solamente se calcula teniendo en cuenta sus propiedades width y height. El margen, los rellenos y los bordes establecidos a un elemento determinar la anchura y la altura final.

Código de ejemplo

                    
                        div{
                            width: 300px;
                            padding-left: 50px;
                            padding-right: 50px;
                            margin-left: 30px;
                            margin-right: 30px;
                            border: 10px solid black;
                        }

La anchura total no son los 300px indicados en la propiedad width, sino que se tiene en cuenta todos sus márgenes, rellenos, bordes.

De esta forma, la anchura del elemento en pantalla sería igual a la suma de la anchura original, los margenes, los bordes y los rellenos.

30px + 10px + 50px + 300px + 50px + 10px + 30px = 480 píxel

Modo de compatibilidad Interner explorer

El mode de compatibilidad de la página se indica mediante la etiqueta <meta> con la propiedad X-UA-compatible y cuyo valor es el que utiliza Internet Exprorer 8 para determinar el modo que se utiliza.

                    
                        <!-- Modo IE5 (Internet Explorer 5) -->
                        <Head>
                        <meta http-equiv="X-UA-Compatible" content="IE=5">
                        ...
                        </Head>

                        <!-- Modo IE7 (Internet Explorer 7) -->
                        <Head>
                        <meta http-equiv="X-UA-Compatible" content="IE=7">
                        ...
                        </Head>

                        <!-- Modo IE8 (Internet Explorer 8) -->
                        <Head>
                        <meta http-equiv="X-UA-Compatible" content="IE=8">
                        ...
                        </Head>

                        <!-- Emular el Modo IE7 (Internet Explorer 7) -->
                        <Head>
                        <meta http-equiv="X-UA-Compatible" content="IE=EmulateIE7">
                        ...
                        </Head>

                        <!-- Emular el Modo IE8 (Internet Explorer 8) -->
                        <Head>
                        <meta http-equiv="X-UA-Compatible" content="IE=EmulateIE8">
                        ...
                        </Head>

                        <!-- Modo límite -->
                        <Head>
                        <meta http-equiv="X-UA-Compatible" content="IE=edge">
                        ...
                        </Head>

Este metodo es aplicable para paginas antiguas.

Fondos

El ultimo elemento que froma el box model (modelo de caja) es el fondo de la caja del elemento. El fondo puede ser un color simple o una imagen. el fondo solamente es visible en el área ocupada por el contenido y su relleno(padding), ya que el color de los bordes(border) se controla direcetamente desde los bordes y la zonda de los márgenes siempres son trasparentes.

CSS define 5 propiedades para establecer el fondo de cada elemento (background-color, background-image, background-repeat, background-attachment, background-position) y otra propiedad de tipo "shorthand" (background).

background-color Color de fondo
Valores <color> | trasparent | inherit
Se aplica a Todos los elementos
Valor inicial trasparent
Descripción Establece un color de fondo para los elementos

Ejemplo.

                    
                        body{
                            background-color: #F5F5F5;
                        }

background-image

En ocaciones es necesario crear un fondo mas complejo que un simple color. CCS permite mostrar una imagen como fondo de cualquier elemento.

background-image imagen de fondo
Valores <utl> | none | inherit
Se aplica a Todos los elementos
Valor inicial none
Descripción Establece una imagen como fondo a todos los elementos

Nota: Em CSS permite establcer de forma simultánea un color y una imagen de fondo. En este caso, la imagen se muestra adelante color.

Ejemplo de Código

                    
                        body {
                            background-image: url ("imagenes/fondos.png")
                        }

Suele habitual indicar un color de fondo siempre que se muestre una imagen de fondo. En caso que no se pueda mostrar o contenga error, el navegador mostrara el color indicado.

Si la imagen que se quiere mostrar es demaciado grande para el fodo del elemento, solamente se muestra la parte comprendida en el tamaño del elemento. Si la imagen es mas peqieña que el elemento, CSS la repite horizontalmente y verticalmente asta llenar el fondo del elemento.

background-repeat

En ocaciones, no es conveniente que la imagen se repita horizontal y verticalmente. Para ello, CSS introduce la propiedad background-repeat que permite controlar la forma de repetición de la imágenes de fondo.

background-repeat repetición de la imagen de fondo
Valores repeat | repeat-x | repeat-y | no-repeat | inherit
Se aplica a Todos los elementos
Valor inicial repeat
Descripción Controla la forma en que se repite las imágenes de fondo

El valor repeat indica que la imagen se deve repetir en todas las direcciones por defecto y por tanto, es el comportamiento por defecto. El valor no-repeat muestra una sola imagen y no se repite en ninguna direscción. El valor repeat-x repite la imagen sólo horizontalmente y el valor repeat-y repite la imagen solo en vertical.

Ejemplos

Código de ejemplo.

                    
                        div{
                            background: url("/imgen/ds.png") repeat-x;
                            width: 100%;
                            text-aling: center;
                        }

background-position

Ademas de seleccionar el tipo de repetición de la imágenes de fondo, CSS permite controlar la posición de la imagen dentro del fondo del elemento mediante la propiedad background-position

background-position posición de la imagen de fondo.
Valores ((<porcentaje> | <medida> | left | center |
tight) (<porcentaje> | <medida> | top |
center | bottom)?) | ((left | center |
right) || (top | center | bottom)) |
inherit
Se aplica a Todos los elementos
Valor inicial 0% 0%
Descripción Controla la posición en la que se muestra la imagen en el
fondo del elemento.

Si se indica dos porcetajes o dos medidas, el primero indica el desplazamiento horizontal y el segundo el desplazamiento vertical respecto del origen (situado en la esquina superior izquierda). Si solamente se indica un porcentaje o una medida, se considera horizotal y al desplazamiento vertical se le asigna automáticamente el valor de 50%.

Si el valor de la propiedad background-position se indica mediante dos porcentajes x%, y% el navagador coloca el punto (x%, y%) de la imagen de fondo en el punto (x%, y%) del elemento.

Las palabras claves permitidas son equivalentes a algunos porcentajes significativos: top = 0%, left = 0%, center = 50% bottom = 100%, right = 100%.

CSS permite mezclar porcentajes y palabras claves, como por ejemplo 50% 2cm, center 2cm, center 10%.

Ejemplos

background-attachment

background-attachment Comportamiento de la imagen de fondo
Valores scroll | fixed | inherit
Se aplica a Todos los elementos
Valor inicial scroll
Descripción Controla la forma en la que se visualiza la imagen de
fondo: permanece fija cuando se hace scrill en la ventana
del navegador o se desplaza junto con la ventana.

Para que una imagen de fondon se muestre fija al desplazar la ventana del navegador, se deve añadir la propiedad background-attachment: fixed

background

Por último, CSS define una propiedad de tipo "shorthand" par indicar todas las propiedades de los colores e imágenes de fondo de forma directa.

background Fondo de un elemento.
Valores (<background-color> || <background-image> ||
<background-repeat> || <background-attachment> ||
<background-position>) | inherit
Se aplica a Todos los elementos
Valor inicial -
Descripción Establece todas las propiedades del fondo de un elemento

El orden en el que se indican las propiedades es indiferente, aunque en general se sigue el formato indicado de color, url de imagen, repetición y posición
