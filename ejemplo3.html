Importar CSS desde la etiqueta style.

Forma altenativa de incluir un archivo externo.

En este caso, para incluir en la página HTML los estilos definidos en archivos CSS externos se utiliza una regla especial de tipo @import. Las reglas de tipo @import siempre preceden a cuaquier otra regla CSS(con la única excepción de la regla @charset).

La URL del archivo CSS externo se indica mediante una cadena de texto encerrada con comillas simples o dobles o mediante la palabra reservada url(). De esta forma,las siguientes reglas @import son equivalentes:

@import '/styles/style1.css'
@import "/styles/style1.css"
@import url('/styles/style1.css')
@import url("/styles/style1.css")

Icluir CSS en elementos HTML

Este método para incluir estilos CSS en un documento HTML es el peor y el menos utilizado, ya que tiene los mismos problemas que la utilización de las etiquetas font.

Esta forma de incluir CSS directamente en los elementos HTML solamente se utiliza en determinadas situaciones en las que se debe incluir un estilo muy específico para un solo elemento concreto.
